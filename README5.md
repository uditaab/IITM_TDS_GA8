# Customer Support Response Time Analysis

This repository contains a professional Seaborn violinplot visualization
that analyzes the distribution of customer support response times across
different communication channels.

**Prepared by:** 24ds3000072@ds.study.iitm.ac.in

## Files Included

- `chart.py` — Python script generating the visualization
- `chart.png` — 512x512 violinplot image
- `README.md` — Project documentation

## Description

The analysis simulates realistic response-time behavior for four channels:

- Email — slowest and most variable
- Chat — fastest and most consistent
- Phone — moderate speed
- Social Media — moderate variability

The generated visualization follows Seaborn best practices for
executive-level reporting.
